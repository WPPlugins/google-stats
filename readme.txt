== Installation ==

1. Upload the Google Analytics 'google-stats' directory to the '/wp-content/plugins/' directory
2. Activate the Google Analytics plugin through the 'Plugins' menu in WordPress
3. Configure the Google Analytics Property ID (UA-12345678-0) into the settings. (Admin > Settings > Google Analytics)
